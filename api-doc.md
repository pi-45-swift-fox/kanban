## POST /login
### request
- body
```json
{
    "email":"anton@mail.com",
    "password":"123456"
}
```
### response 200
- body
```json
{
    "access_token" :"iwejij323ji3jf"
}
```
### response 400
- body
```json
{
    "message":"User or Password Incorrect"
}
```

### response 500
- body
```json
{
    "message":"internal server error"
}
```
